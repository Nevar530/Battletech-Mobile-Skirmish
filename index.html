      Controls Help
      
        
      
      
      
      
        
      
          Map & Camera
      
          Mouse: Wheel = Zoom · Left-drag (Space held) = Pan · Middle-drag = Pan
      
          Keyboard: Arrows = Pan · 0 = Recenter · F = Fullscreen
      
          Touch: Pinch = Zoom · Two-finger drag = Pan

      

          Terrain Tools
      
          Use left drawer modes. Alt+Click to eyedrop.

      

          LOS (L), Measure: top bar toggles.

      

          Tokens
      
          N = Add · Q/E = Rotate · +/- = Size · [ / ] = Team · Enter = Rename · Delete = Remove

      

          Export/Import: Top bar JSON buttons.
